## ğŸ’¡ Key Concepts

### API Service Pattern
- Centralized API management
- Single source of truth
- Automatic authentication
- Consistent error handling

### Authentication Flow
1. User registers/logs in
2. Backend returns JWT token
3. Token stored in localStorage
4. Token automatically added to API requests
5. Protected routes validate token
6. Invalid tokens trigger logout

### Data Flow
1. React component calls API function
2. API service adds auth header and URL
3. Fetch sends HTTP request to backend
4. Backend processes and responds with JSON
5. API service checks response and returns data
6. Component updates state and UI re-renders
